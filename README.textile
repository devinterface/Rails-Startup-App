h1. Rails Startup Template

This template lets you to create quickly new Rails applications using Rails 2.3. To use it, just specify the -m switch when creating a Rails application:

<pre><code>
rails new_app_name -m http://github.com/devinterface/Rails-Startup-App/raw/master/rails_startup.rb
</code></pre>

h2. Generated Application

Here's a list of what this template sets up:

h3. SCM
* git repository

h3. Javascript
* Choice of "Prototype":http://www.prototypejs.org or "jQuery":http://jquery.com

h3. CSS
* "Blueprint":http://github.com/joshuaclayton/blueprint-css/tree/master as css framework

h3. Rails
* Choice of "Authlogic":http://github.com/binarylogic/authlogic/tree/master with or without "OpenID":http://github.com/binarylogic/authlogic_openid/tree/master support (includes models, controllers and views)

h3. Testing
* BDD testing support using "RSpec":http://github.com/dchelimsky/rspec/tree/master, "Cucumber":http://github.com/aslakhellesoy/cucumber/tree/master, and "FactoryGirl":http://github.com/thoughtbot/factory_girl/tree/master
